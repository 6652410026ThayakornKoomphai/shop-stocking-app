##  📦 Shop Stocking App
แอปพลิเคชันสำหรับจัดการสต็อกสินค้าและเปรียบเทียบราคาสินค้า พัฒนาด้วย React Native ผ่าน Expo และใช้ Supabase เป็นระบบหลังบ้าน (Backend) เพื่อช่วยให้การจัดซื้อหรือเช็คของในร้านค้าเป็นเรื่องง่ายและเป็นระบบมากขึ้น

##  ✨ คุณสมบัติเด่น (Features)
  * Authentication System: ระบบสมัครสมาชิกและเข้าสู่ระบบผ่าน Supabase Auth
  * Inventory Management: สร้างและจัดการรายการสต็อกสินค้า แยกตามรายการหรือหมวดหมู่ที่ต้องการ
  * Price Comparison: ฟีเจอร์เปรียบเทียบราคาสินค้า เพื่อช่วยในการตัดสินใจเลือกซื้อสินค้าที่คุ้มค่าที่สุด
  * Real-time Database: ข้อมูลซิงค์อัตโนมัติผ่าน Supabase ทำให้ข้อมูลเป็นปัจจุบันเสมอ
  * Responsive UI: ออกแบบด้วย NativeWind (Tailwind CSS) ทำให้หน้าตาสวยงามและใช้งานง่ายทั้งบน iOS และ Android

##  🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

  * Styling: NativeWind (Tailwind CSS for React Native)
  * Database & Auth: Supabase
  * Language: TypeScript
